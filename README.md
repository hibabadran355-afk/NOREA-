# NOREA-

Einföld vefsíða — dæmi

Þetta verkefni inniheldur grunnsnið fyrir vefsíðu byggða með HTML, CSS og JavaScript.

Skrár sem bætt var við:

- `index.html` — aðalsíða
- `styles.css` — stílar
- `script.js` — einfalt JavaScript (hnappur, dagsetning, þema)

Hvernig prófa:

1. Opna `index.html` beint í vafra (tvöfaldur smellur eða "Open File" í editor).
2. Eða keyra einfaldan vefþjón í verkefnamöppu og opna http://localhost:8000:

```bash
python3 -m http.server 8000
```

3. Sjá og prófa hnappa: "Segðu mér halló" og "Skipta um þema".

Vinna lokið: einfalt upphaf, hægt að bæta við fleiri síðum, forms eða API-samskiptum síðar.
